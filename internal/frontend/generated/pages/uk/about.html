Про securememo.app

securememo.app — це додаток, орієнтований на конфіденційність, який дозволяє створювати зашифровані записки, що самознищуються після прочитання або закінчення терміну. Побудовано з акцентом на безпеку та конфіденційність. Вихідний код доступний на GitHub.

Технологічний стек

  • Go service: Self-hosted HTTP service
  • SQLite database: SQLite-powered database for secure storage
  • Web Crypto API: Web Crypto API: Шифрування AES-256 на стороні клієнта
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Сучасний, адаптивний веб-інтерфейс
  • Вихідний код доступний на GitHub: Переглянути вихідний код на GitHub

Функції безпеки

🔐 Шифрування на стороні клієнта

Весь процес шифрування відбувається у вашому браузері через Web Crypto API. Ваша записка шифрується AES-256 перед відправкою на сервер.

🔑 Окремий обмін паролем

Пароль для шифрування генерується окремо від URL і має бути переданий іншим каналом для підвищення безпеки.

💥 Самознищувані записки

Після прочитання або закінчення терміну записка негайно та остаточно видаляється з нашої бази даних. Жодних слідів не залишається.

🌐 Без зберігання паролів

Ми ніколи не зберігаємо паролі для шифрування на наших серверах. Вони існують лише у пам’яті вашого браузера.

Self-hosted service

Runs as a Go service backed by SQLite, suitable for deployment behind a reverse proxy or tunnel.

🔒 Конфіденційність понад усе

Не потрібні облікові записи користувачів. Без відстеження. Без аналітики. Ваша конфіденційність — наш пріоритет.

Як користуватися

1. Створіть записку

Перейдіть до "Створити захищену записку" і введіть свою записку. Виберіть час зберігання (8 год, 1 день, 2 дні, 1 тиждень або максимум 30 днів). Записки видаляються після прочитання або закінчення терміну. Система згенерує випадковий пароль для шифрування.

2. Поділіться посиланням і паролем

Скопіюйте згенерований URL і пароль, а потім передайте їх отримувачу через різні канали для підвищення безпеки.

3. Самознищення

Коли отримувач відкриє посилання і введе пароль окремо, записка буде розшифрована і остаточно видалена, якщо прочитана, або автоматично видалена, якщо закінчився термін. Прострочені записки автоматично очищаються щогодини.
