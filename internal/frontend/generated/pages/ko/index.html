securememo.app

읽거나 만료된 후 자체 삭제되는 암호화된 메모를 만드세요. 비밀은 안전하게 보호됩니다.

보안 메모 작성 더 알아보기

작동 방식

🔐 메모를 입력

통제권은 사용자에게 있습니다. 텍스트는 기기에서 암호화되며 평문 버전이 브라우저를 절대 벗어나지 않습니다.

🔗 안전하게 공유

고유한 URL과 별도의 강력한 비밀번호를 받고, 추가 보안을 위해 다른 채널을 통해 공유하세요.

💥 자동 삭제

메모는 읽힌 후 또는 시간 제한이 만료되면(8시간~30일) 자동으로 영구 삭제됩니다.

보안 기능

🔒 진정한 클라이언트 측 암호화

메모는 브라우저 내에서 AES-256-GCM과 강력한 키 유도로 암호화됩니다. 저희가 평문을 볼 일은 없습니다.

🗑️ 신뢰할 수 있는 자동 삭제

메모는 읽거나 만료된 후 매시간 자동 정리로 영원히 사라집니다. 백업이나 복구는 불가능합니다.

🌐 비밀번호 무지식 원칙

브라우저에서 완전히 비밀번호를 생성하고 처리합니다. 서버로 전송되거나 저장되지 않습니다.
