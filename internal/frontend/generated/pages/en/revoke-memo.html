Before you continue:

  • This deletes the memo immediately if it has not already been read, expired, or deleted
  • Deletion is permanent and the memo cannot be recovered
  • The recipient will no longer be able to read the memo
  • This request sends only the memo ID and revoke token to the server. It does not send the memo password or plaintext memo contents.
