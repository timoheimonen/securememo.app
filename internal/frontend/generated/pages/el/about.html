Σχετικά με το securememo.app

Το securememo.app είναι μια εφαρμογή με έμφαση στην ιδιωτικότητα που σας επιτρέπει να δημιουργείτε κρυπτογραφημένες σημειώσεις που αυτοκαταστρέφονται μετά την ανάγνωση ή τη λήξη. Χτισμένο με προτεραιότητα στην ασφάλεια και την ιδιωτικότητα. Ο πηγαίος κώδικας είναι διαθέσιμος στο GitHub.

Τεχνολογίες

  • Cloudflare Workers: Cloudflare Workers: Πλατφόρμα serverless για παγκόσμια απόδοση
  • D1 Database: D1 Database: Βάση δεδομένων SQLite για ασφαλή αποθήκευση
  • Web Crypto API: Web Crypto API: Κρυπτογράφηση AES-256 στην πλευρά του πελάτη
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Σύγχρονο, ανταποκρινόμενο web περιβάλλον
  • Ο πηγαίος κώδικας είναι διαθέσιμος στο GitHub: Δείτε τον πηγαίο κώδικα στο GitHub

Χαρακτηριστικά Ασφαλείας

🔐 Κρυπτογράφηση στην Πλευρά του Πελάτη

Όλη η κρυπτογράφηση γίνεται στον περιηγητή σας μέσω του Web Crypto API. Η σημείωσή σας κρυπτογραφείται με AES-256 πριν σταλεί στους διακομιστές μας.

🔑 Ξεχωριστή Κοινοποίηση Κωδικού

Ο κωδικός κρυπτογράφησης δημιουργείται ξεχωριστά από το URL και πρέπει να κοινοποιείται μέσω διαφορετικού καναλιού για αυξημένη ασφάλεια.

💥 Αυτοκαταστρεφόμενες Σημειώσεις

Μόλις μια σημείωση διαβαστεί ή λήξει, διαγράφεται άμεσα και οριστικά από τη βάση δεδομένων μας. Δεν μένουν ίχνη.

🌐 Χωρίς Αποθήκευση Κωδικών

Δεν αποθηκεύουμε ποτέ τους κωδικούς κρυπτογράφησης στους διακομιστές μας. Υπάρχουν μόνο στη μνήμη του περιηγητή σας.

⚡ Παγκόσμια Απόδοση

Χτισμένο στο παγκόσμιο δίκτυο της Cloudflare για αστραπιαία πρόσβαση από οπουδήποτε στον κόσμο.

🔒 Ιδιωτικότητα Πάνω Απ’ Όλα

Δεν απαιτούνται λογαριασμοί χρήστη. Χωρίς παρακολούθηση. Χωρίς analytics. Η ιδιωτικότητά σας είναι προτεραιότητά μας.

Πώς να Χρησιμοποιήσετε

1. Δημιουργία Σημείωσης

Μεταβείτε στη "Δημιουργία Ασφαλούς Σημείωσης" και πληκτρολογήστε τη σημείωσή σας. Επιλέξτε χρόνο λήξης (8 ώρες, 1 ημέρα, 2 ημέρες, 1 εβδομάδα ή μέγιστο 30 ημέρες). Οι σημειώσεις διαγράφονται όταν διαβαστούν ή λήξουν. Το σύστημα θα δημιουργήσει έναν τυχαίο κωδικό κρυπτογράφησης.

2. Κοινοποίηση Συνδέσμου και Κωδικού

Αντιγράψτε το παραγόμενο URL και τον κωδικό, και κοινοποιήστε τα στον παραλήπτη σας μέσω ξεχωριστών καναλιών για αυξημένη ασφάλεια.

3. Αυτοκαταστροφή

Όταν ο παραλήπτης ανοίξει τον σύνδεσμο και εισάγει τον κωδικό ξεχωριστά, η σημείωση θα αποκρυπτογραφηθεί και θα διαγραφεί οριστικά αν διαβαστεί, ή θα διαγραφεί αυτόματα αν λήξει. Οι ληγμένες σημειώσεις καθαρίζονται αυτόματα κάθε 1 ώρα.
