Om securememo.app

securememo.app er en privatlivsfokuseret applikation, der lader dig oprette krypterede notater, som selvdestruerer efter de er læst eller udløbet. Bygget med sikkerhed og privatliv som højeste prioritet. Kildekoden er tilgængelig på GitHub.

Teknologisk Fundament

  • Cloudflare Workers: Cloudflare Workers: Serverløs platform for global ydeevne
  • D1 Database: D1 Database: SQLite-baseret database til sikker lagring
  • Web Crypto API: Web Crypto API: Klientbaseret AES-256 kryptering
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Moderne, responsivt webinterface
  • Kildekode tilgængelig på GitHub: Se kildekoden på GitHub

Sikkerhedsfunktioner

🔐 Klientbaseret kryptering

Al kryptering sker i din browser via Web Crypto API. Dit notat krypteres med AES-256 før det sendes til vores servere.

🔑 Separat adgangskodedeling

Krypteringsadgangskoden genereres separat fra URLen og bør deles via en anden kanal for øget sikkerhed.

💥 Selvdestruerende notater

Når et notat er læst eller udløbet, slettes det straks og permanent fra vores database. Ingen spor tilbage.

🌐 Ingen adgangskodelagring

Vi gemmer aldrig krypteringsadgangskoder på vores servere. De eksisterer kun i din browsers hukommelse.

⚡ Global ydeevne

Bygget på Cloudflares globale netværk for lynhurtig adgang fra hele verden.

🔒 Privatliv først

Ingen brugeroprettelse krævet. Ingen sporing. Ingen analyse. Dit privatliv er vores prioritet.

Sådan bruges

1. Opret et notat

Gå til "Opret Sikkert Notat" og skriv dit notat. Vælg en udløbstid (8t, 1d, 2d, 1 uge eller maks. 30 dage). Notater slettes når de er læst eller udløbet. Systemet genererer en tilfældig krypteringsadgangskode.

2. Del link og adgangskode

Kopiér den genererede URL og adgangskode, og del dem med din modtager via separate kanaler for øget sikkerhed.

3. Selvdestruktion

Når modtageren åbner linket og indtaster adgangskoden separat, dekrypteres notatet og slettes derefter permanent, hvis det er læst, eller slettes automatisk, hvis det er udløbet. Udløbne notater ryddes automatisk op hver time.
