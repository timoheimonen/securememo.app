securememo.app के बारे में

securememo.app एक गोपनीयता-केंद्रित एप्लिकेशन है जो आपको एन्क्रिप्टेड मेमो बनाने की अनुमति देती है जो पढ़ने या समाप्ति के बाद स्व-विनाश हो जाते हैं। सुरक्षा और गोपनीयता को सर्वोच्च प्राथमिकता के रूप में बनाया गया। सोर्स कोड GitHub पर उपलब्ध है।

प्रौद्योगिकी स्टैक

  • Go service: Self-hosted HTTP service
  • SQLite database: SQLite-powered database for secure storage
  • Web Crypto API: Web Crypto API: क्लाइंट-साइड AES-256 एन्क्रिप्शन
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: आधुनिक, रिस्पॉन्सिव वेब इंटरफेस
  • सोर्स कोड GitHub पर उपलब्ध है: GitHub पर सोर्स कोड देखें

सुरक्षा सुविधाएं

🔐 क्लाइंट-साइड एन्क्रिप्शन

सभी एन्क्रिप्शन Web Crypto API का उपयोग करके आपके ब्राउज़र में होता है। आपका मेमो हमारे सर्वर पर भेजे जाने से पहले AES-256 के साथ एन्क्रिप्ट होता है।

🔑 अलग पासवर्ड साझाकरण

एन्क्रिप्शन पासवर्ड URL से अलग से जेनरेट होता है और बेहतर सुरक्षा के लिए इसे अलग चैनल के माध्यम से साझा किया जाना चाहिए।

💥 स्व-विनाशकारी मेमो

एक बार मेमो पढ़ा या समाप्त हो जाने पर, यह तुरंत और स्थायी रूप से हमारे डेटाबेस से डिलीट हो जाता है। कोई निशान नहीं बचता।

🌐 कोई पासवर्ड स्टोरेज नहीं

हम कभी भी एन्क्रिप्शन पासवर्ड को अपने सर्वर पर स्टोर नहीं करते। वे केवल आपके ब्राउज़र की मेमोरी में मौजूद रहते हैं।

Self-hosted service

Runs as a Go service backed by SQLite, suitable for deployment behind a reverse proxy or tunnel.

🔒 गोपनीयता प्राथमिक

कोई यूजर अकाउंट की आवश्यकता नहीं। कोई ट्रैकिंग नहीं। कोई एनालिटिक्स नहीं। आपकी गोपनीयता हमारी प्राथमिकता है।

उपयोग कैसे करें

1. मेमो बनाएं

"सुरक्षित मेमो बनाएं" पर जाएं और अपना मेमो टाइप करें। समाप्ति समय चुनें (8 घंटे, 1 दिन, 2 दिन, 1 सप्ताह या अधिकतम 30 दिन)। मेमो पढ़े जाने या समाप्त होने पर डिलीट हो जाते हैं। सिस्टम एक रैंडम एन्क्रिप्शन पासवर्ड जेनरेट करेगा।

2. लिंक और पासवर्ड साझा करें

जेनरेट किए गए URL और पासवर्ड को कॉपी करें, फिर बेहतर सुरक्षा के लिए उन्हें अपने प्राप्तकर्ता के साथ अलग चैनलों के माध्यम से साझा करें।

3. स्व-विनाश

जब प्राप्तकर्ता लिंक खोलता है और अलग से पासवर्ड दर्ज करता है, तो मेमो डिक्रिप्ट हो जाएगा और फिर पढ़े जाने पर स्थायी रूप से डिलीट हो जाएगा, या समाप्त होने पर स्वचालित रूप से डिलीट हो जाएगा। समाप्त मेमो हर 1 घंटे में स्वचालित रूप से साफ हो जाते हैं।

शुरू करने के लिए तैयार हैं?

अब अपना पहला सुरक्षित मेमो बनाएं और वास्तविक एंड-टू-एंड एन्क्रिप्शन का अनुभव करें।
