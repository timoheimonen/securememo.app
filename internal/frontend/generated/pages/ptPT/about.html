Sobre o securememo.app

O securememo.app é uma aplicação focada na privacidade que permite criar memos criptografados que se autodestroem após serem lidos ou expirarem. Construído com segurança e privacidade como principais prioridades. Código fonte disponível no GitHub.

Stack de Tecnologia

  • Go service: Self-hosted HTTP service
  • SQLite database: SQLite-powered database for secure storage
  • Web Crypto API: Web Crypto API: Criptografia AES-256 do lado do cliente
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Interface web moderna e responsiva
  • Código fonte disponível no GitHub: Ver código fonte no GitHub

Recursos de Segurança

🔐 Criptografia do Lado do Cliente

Toda criptografia acontece no seu navegador usando a Web Crypto API. Seu memo é criptografado com AES-256 antes de ser enviado para nossos servidores.

🔑 Partilha Separada de Palavra-passe

A palavra-passe de criptografia é gerada separadamente da URL e deve ser partilhada por um canal diferente para segurança aprimorada.

💥 Memos Autodestrutivos

Uma vez que um memo é lido ou expira, ele é imediatamente e permanentemente excluído do nosso banco de dados. Nenhum rastro permanece.

🌐 Nenhum Armazenamento de Palavra-passe

Nunca armazenamos palavras-passe de criptografia nos nossos servidores. Existem apenas na memória do seu navegador.

Self-hosted service

Runs as a Go service backed by SQLite, suitable for deployment behind a reverse proxy or tunnel.

🔒 Privacidade em Primeiro

Nenhuma conta de utilizador necessária. Sem rastreamento. Sem análises. A sua privacidade é a nossa prioridade.

Como Usar

1. Criar um Memo

Vá para "Criar Memo Seguro" e introduza o seu memo. Escolha um tempo de expiração (8h, 1d, 2d, 1 semana ou máximo de 30 dias). Memos são excluídos quando lidos ou expirados. O sistema gerará uma palavra-passe de criptografia aleatória.

2. Compartilhar o Link e a Senha

Copie a URL e palavra-passe geradas, e partilhe-as com o seu destinatário por canais separados para segurança aprimorada

3. Autodestruição

Quando o destinatário abrir o link e introduzir a palavra-passe separadamente, o memo será descriptografado e então permanentemente excluído se lido, ou excluído automaticamente se expirado. Memos expirados são automaticamente limpos a cada 1 hora.
