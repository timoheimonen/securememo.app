securememo.appについて

securememo.appは、読み取りまたは期限切れ後に自己破壊する暗号化メモを作成できるプライバシー重視のアプリケーションです。セキュリティとプライバシーを最優先に構築されています。ソースコードはGitHubで利用可能です。

技術スタック

  • Go service: Go service: グローバルに高速な提供を実現するサーバーレス実行基盤
  • SQLite database: SQLiteデータベース: 安全なストレージのためのSQLiteベースのデータベース
  • Web Crypto API: Web Crypto API: クライアントサイドAES-256暗号化
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: モダンでレスポンシブなWebインターフェース
  • ソースコードはGitHubで利用可能: GitHubでソースコードを見る

セキュリティ機能

🔐 クライアントサイド暗号化

すべての暗号化はWeb Crypto APIを使用してブラウザで行われます。メモはサーバーに送信される前にAES-256で暗号化されます。

🔑 別々のパスワード共有

暗号化パスワードはURLとは別に生成され、セキュリティを強化するために異なるチャネルを通じて共有する必要があります。

💥 自己破壊メモ

メモが読み取られるか期限切れになると、データベースから即座に完全に削除されます。痕跡は残りません。

🌐 パスワード保存なし

サーバーに暗号化パスワードを保存することはありません。ブラウザのメモリにのみ存在します。

⚡ グローバルパフォーマンス

世界中のどこからでも超高速アクセスを実現するthe hosting providerのグローバルネットワーク上に構築。

🔒 プライバシー第一

ユーザーアカウント不要。追跡なし。アナリティクスなし。あなたのプライバシーが私たちの優先事項です。

使用方法

1. メモを作成

「セキュアメモを作成」に移動してメモを入力します。有効期限（8時間、1日、2日、1週間、または最大30日）を選択します。メモは読み取られるか期限切れになると削除されます。システムがランダムな暗号化パスワードを生成します。

2. リンクとパスワードを共有

生成されたURLとパスワードをコピーし、セキュリティを強化するために別々のチャネルを通じて受信者と共有します。

3. 自己破壊

受信者がリンクを開いて別途パスワードを入力すると、メモが復号化され、読み取られた場合は完全に削除されるか、期限切れの場合は自動的に削除されます。期限切れのメモは1時間ごとに自動的にクリーンアップされます。
