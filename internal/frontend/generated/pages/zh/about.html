关于 securememo.app

securememo.app 是一个以隐私为重点的应用程序，允许您创建在阅读或过期后自毁的加密备忘录。以安全和隐私为首要优先级构建。源代码在 GitHub 上可用。

技术栈

  • Go service: Go service：用于全球性能的无服务器计算平台
  • SQLite database: SQLite 数据库：由 SQLite 提供支持的安全存储数据库
  • Web Crypto API: Web Crypto API：客户端 AES-256 加密
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript：现代、响应式网络界面
  • 源代码在 GitHub 上可用： 在 GitHub 上查看源代码

安全功能

🔐 客户端加密

所有加密都在您的浏览器中使用 Web Crypto API 进行。您的备忘录在发送到我们的服务器之前会使用 AES-256 进行加密。

🔑 单独密码分享

加密密码与 URL 分开生成，应通过不同的渠道分享以增强安全性。

💥 自毁备忘录

一旦备忘录被阅读或过期，它会立即从我们的数据库中永久删除。不会留下任何痕迹。

🌐 不存储密码

我们从不在服务器上存储加密密码。它们只存在于您浏览器的内存中。

⚡ 全球性能

基于 the hosting provider 的全球网络构建，从世界任何地方都能闪电般访问。

🔒 隐私优先

无需用户账户。无跟踪。无分析工具。您的隐私是我们的首要考虑。

如何使用

1. 创建备忘录

转到“创建安全备忘录”并输入您的备忘录。选择过期时间（8小时、1天、2天、1周或最多30天）。备忘录在阅读或过期时会被删除。系统将生成一个随机加密密码。

2. 分享链接和密码

复制生成的 URL 和密码，然后通过单独的渠道与您的收件人分享，以增强安全性。

3. 自毁

当收件人打开链接并单独输入密码时，备忘录将被解密，然后在阅读后永久删除，或在过期后自动删除。过期的备忘录每小时自动清理一次。
