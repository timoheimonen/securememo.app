О securememo.app

securememo.app — это приложение, ориентированное на приватность, позволяющее создавать зашифрованные заметки, которые самоуничтожаются после прочтения или истечения срока. Безопасность и приватность — наш приоритет. Исходный код доступен на GitHub.

Технологии

  • Go service: Self-hosted HTTP service
  • SQLite database: SQLite-powered database for secure storage
  • Web Crypto API: Web Crypto API: клиентское шифрование AES-256
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: современный, адаптивный веб-интерфейс
  • Исходный код доступен на GitHub: Посмотреть исходный код на GitHub

Функции безопасности

🔐 Клиентское шифрование

Всё шифрование происходит в вашем браузере с помощью Web Crypto API. Ваша заметка шифруется с помощью AES-256 до отправки на сервер.

🔑 Отдельная передача пароля

Пароль для шифрования генерируется отдельно от ссылки и должен передаваться по другому каналу для повышения безопасности.

💥 Самоуничтожающиеся заметки

После прочтения или истечения срока заметка немедленно и навсегда удаляется из нашей базы данных. Следов не остаётся.

🌐 Без хранения паролей

Мы никогда не храним пароли для шифрования на наших серверах. Они существуют только в памяти вашего браузера.

Self-hosted service

Runs as a Go service backed by SQLite, suitable for deployment behind a reverse proxy or tunnel.

🔒 Приватность прежде всего

Не требуется аккаунт. Нет отслеживания. Нет аналитики. Ваша приватность — наш приоритет.

Как пользоваться

1. Создайте заметку

Перейдите на "Создать защищённую заметку" и введите свою заметку. Выберите срок хранения (8ч, 1д, 2д, 1 неделя или максимум 30 дней). Заметки удаляются после прочтения или истечения срока. Система сгенерирует случайный пароль для шифрования.

2. Передайте ссылку и пароль

Скопируйте сгенерированную ссылку и пароль, затем передайте их получателю через разные каналы для повышения безопасности.

3. Самоуничтожение

Когда получатель откроет ссылку и введёт пароль отдельно, заметка будет расшифрована и навсегда удалена, если прочитана, либо удалена автоматически при истечении срока. Просроченные заметки автоматически очищаются каждый час.
