เกี่ยวกับ securememo.app

securememo.app เป็นแอปที่เน้นความเป็นส่วนตัว ช่วยให้คุณสร้างบันทึกเข้ารหัสที่ทำลายตัวเองหลังจากถูกอ่านหรือหมดอายุ พัฒนาด้วยความปลอดภัยและความเป็นส่วนตัวเป็นอันดับแรก มีซอร์สโค้ดบน GitHub

เทคโนโลยีที่ใช้

  • Go service: Self-hosted HTTP service
  • SQLite database: SQLite-powered database for secure storage
  • Web Crypto API: Web Crypto API: การเข้ารหัส AES-256 ฝั่งไคลเอนต์
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: ส่วนติดต่อเว็บที่ทันสมัยและตอบสนองไว
  • มีซอร์สโค้ดบน GitHub: ดูซอร์สโค้ดบน GitHub

คุณสมบัติด้านความปลอดภัย

🔐 การเข้ารหัสฝั่งไคลเอนต์

การเข้ารหัสทั้งหมดเกิดขึ้นในเบราว์เซอร์ของคุณโดยใช้ Web Crypto API บันทึกของคุณถูกเข้ารหัสด้วย AES-256 ก่อนส่งไปยังเซิร์ฟเวอร์ของเรา

🔑 การแชร์รหัสผ่านแยกต่างหาก

รหัสผ่านสำหรับการเข้ารหัสถูกสร้างแยกจาก URL และควรแชร์ผ่านช่องทางที่แตกต่างกันเพื่อเพิ่มความปลอดภัย

💥 บันทึกทำลายตัวเอง

เมื่อบันทึกถูกอ่านหรือหมดอายุ จะถูกลบออกจากฐานข้อมูลของเราในทันทีและถาวร ไม่เหลือร่องรอย

🌐 ไม่จัดเก็บรหัสผ่าน

เราไม่เคยจัดเก็บรหัสผ่านการเข้ารหัสบนเซิร์ฟเวอร์ของเรา รหัสผ่านจะอยู่ในหน่วยความจำของเบราว์เซอร์คุณเท่านั้น

Self-hosted service

Runs as a Go service backed by SQLite, suitable for deployment behind a reverse proxy or tunnel.

🔒 ความเป็นส่วนตัวมาก่อน

ไม่ต้องมีบัญชีผู้ใช้ ไม่มีการติดตาม ไม่มีการวิเคราะห์ ความเป็นส่วนตัวของคุณมาก่อน

วิธีใช้งาน

1. สร้างบันทึก

ไปที่ “สร้างบันทึกลับที่ปลอดภัย” แล้วพิมพ์บันทึกของคุณ เลือกเวลาหมดอายุ (8 ชม., 1 วัน, 2 วัน, 1 สัปดาห์ หรือสูงสุด 30 วัน) บันทึกจะถูกลบเมื่อถูกอ่านหรือหมดอายุ ระบบจะสร้างรหัสผ่านสำหรับการเข้ารหัสแบบสุ่ม

2. แชร์ลิงก์และรหัสผ่าน

คัดลอก URL และรหัสผ่านที่สร้างขึ้น แล้วแชร์กับผู้รับผ่านช่องทางที่แตกต่างกันเพื่อความปลอดภัยที่มากขึ้น

3. ทำลายตัวเอง

เมื่อผู้รับเปิดลิงก์และกรอกรหัสผ่านแยกต่างหาก บันทึกจะถูกถอดรหัสและถูกลบถาวรเมื่ออ่าน หรือถูกลบอัตโนมัติเมื่อหมดอายุ บันทึกที่หมดอายุจะถูกล้างอัตโนมัติทุก 1 ชั่วโมง

พร้อมเริ่มต้นหรือยัง?

สร้างบันทึกที่ปลอดภัยแรกของคุณตอนนี้และสัมผัสการเข้ารหัสปลายทางถึงปลายทางอย่างแท้จริง
