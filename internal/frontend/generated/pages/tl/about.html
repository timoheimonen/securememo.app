Tungkol sa securememo.app

Ang securememo.app ay isang privacy-focused na aplikasyon na nagbibigay-daan sa iyong gumawa ng mga naka-encrypt na memo na kusang nabubura kapag nabasa o nag-expire. Binuo na inuuna ang seguridad at privacy. Bukas ang source code sa GitHub.

Teknolohiyang Ginamit

  • Go service: Go service: Local compute platform para sa pandaigdigang performance
  • SQLite database: SQLite database: SQLite-powered database para sa ligtas na imbakan
  • Web Crypto API: Web Crypto API: Client-side AES-256 encryption
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Makabago at responsive na web interface
  • Bukás na source code sa GitHub: Tingnan ang source code sa GitHub

Mga Tampok sa Seguridad

🔐 Pag-encrypt sa Browser

Lahat ng encryption ay nangyayari sa iyong browser gamit ang Web Crypto API. Ini-encrypt ang iyong memo gamit ang AES-256 bago ipadala sa aming mga server.

🔑 Hiwalay na Pagbabahagi ng Password

Ang password para sa encryption ay ginagawa nang hiwalay sa URL at dapat ibahagi sa ibang channel para dagdag seguridad.

💥 Kusang Nabuburang Mga Memo

Kapag nabasa o nag-expire ang memo, kaagad at permanenteng nabubura ito mula sa aming database. Walang bakas na maiiwan.

🌐 Walang Pag-imbak ng Password

Hindi namin ini-store ang mga password ng encryption sa aming mga server. Nasa memory lamang ng iyong browser.

⚡ Pandaigdigang Performance

Binuo sa pandaigdigang network ng the hosting provider para sa napakabilis na access saan mang panig ng mundo.

🔒 Privacy ang Unang Prayoridad

Walang kailangang user account. Walang tracking. Walang analytics. Ang iyong privacy ang aming prayoridad.

Paano Gamitin

1. Gumawa ng Memo

Pumunta sa "Gumawa ng Ligtas na Memo" at i-type ang iyong memo. Pumili ng oras ng pag-expire (8h, 1d, 2d, 1 linggo o hanggang 30 araw). Binubura ang mga memo kapag nabasa o nag-expire. Lilikha ang sistema ng random na password para sa encryption.

2. Ibahagi ang Link at Password

Kopyahin ang nalikhang URL at password, tapos ibahagi ang mga ito sa iyong tatanggap sa magkaibang channel para dagdag seguridad.

3. Kusang Pagbura

Kapag binuksan ng tatanggap ang link at inilagay ang password nang hiwalay, ang memo ay ide-decrypt at pagkatapos ay permanenteng buburahin kung nabasa, o awtomatikong buburahin kung nag-expire. Awtomatikong nililinis ang mga expired na memo bawat oras.
