securememo.app সম্পর্কে

securememo.app একটি প্রাইভেসি-কেন্দ্রিক অ্যাপ্লিকেশন যেখানে আপনি এনক্রিপ্টেড মেমো তৈরি করতে পারেন যা পড়া বা মেয়াদ শেষ হলে নিজে থেকেই মুছে যায়। সিকিউরিটি ও প্রাইভেসি সর্বোচ্চ অগ্রাধিকার। সোর্স কোড GitHub এ উন্মুক্ত।

প্রযুক্তি স্ট্যাক

  • Cloudflare Workers: Cloudflare Workers: বৈশ্বিক উচ্চ কর্মক্ষমতার সার্ভারলেস কম্পিউট প্ল্যাটফর্ম
  • D1 Database: D1 ডাটাবেস: নিরাপদ স্টোরেজের জন্য SQLite-চালিত ডাটাবেস
  • Web Crypto API: Web Crypto API: ক্লায়েন্ট-সাইড AES-256 এনক্রিপশন
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: আধুনিক, রেসপন্সিভ ওয়েব ইন্টারফেস
  • সোর্স কোড GitHub এ উপলব্ধ: GitHub এ সোর্স কোড দেখুন

নিরাপত্তা বৈশিষ্ট্য

🔐 ক্লায়েন্ট-সাইড এনক্রিপশন

সমস্ত এনক্রিপশন আপনার ব্রাউজারেই Web Crypto API ব্যবহার করে সম্পন্ন হয়। সার্ভারে পাঠানোর আগে আপনার মেমো AES-256 দিয়ে এনক্রিপ্ট হয়।

🔑 পৃথক পাসওয়ার্ড শেয়ারিং

এনক্রিপশন পাসওয়ার্ড URL থেকে আলাদাভাবে তৈরি হয় এবং বাড়তি নিরাপত্তার জন্য ভিন্ন চ্যানেলে শেয়ার করা উচিত।

💥 স্বয়ং-বিনষ্ট মেমো

মেমো পড়া বা মেয়াদ শেষ হওয়ার সাথে সাথেই আমাদের ডাটাবেস থেকে স্থায়ীভাবে মুছে যায়। কোন চিহ্ন থাকে না।

🌐 পাসওয়ার্ড সংরক্ষণ নয়

আমরা কখনোই সার্ভারে এনক্রিপশন পাসওয়ার্ড সংরক্ষণ করি না। এটি শুধু আপনার ব্রাউজারের মেমোরিতে থাকে।

⚡ বৈশ্বিক কর্মক্ষমতা

বিশ্বের যেকোনো স্থান থেকে দ্রুত অ্যাক্সেসের জন্য Cloudflare এর বৈশ্বিক নেটওয়ার্কে নির্মিত।

🔒 প্রাইভেসি প্রথম

কোনো ব্যবহারকারী একাউন্টের প্রয়োজন নেই। কোন ট্র্যাকিং নেই। কোন এনালিটিক্স নেই। আপনার প্রাইভেসি আমাদের অগ্রাধিকার।

কিভাবে ব্যবহার করবেন

১. একটি মেমো তৈরি করুন

"নিরাপদ মেমো তৈরি করুন" এ গিয়ে আপনার মেমো লিখুন। মেয়াদ (৮ ঘন্টা, ১ দিন, ২ দিন, ১ সপ্তাহ বা সর্বোচ্চ ৩০ দিন) নির্বাচন করুন। পড়া বা মেয়াদ শেষ হলে মেমো মুছে যায়। সিস্টেম একটি র্যান্ডম এনক্রিপশন পাসওয়ার্ড তৈরি করবে।

২. লিংক ও পাসওয়ার্ড শেয়ার করুন

জেনারেট হওয়া URL ও পাসওয়ার্ড কপি করে আলাদা চ্যানেলে আপনার প্রাপকের সাথে শেয়ার করুন অতিরিক্ত নিরাপত্তার জন্য।

৩. স্বয়ং-বিনষ্ট

প্রাপক লিংক খুলে আলাদা ভাবে পাওয়া পাসওয়ার্ড দিলে মেমো ডিক্রিপ্ট হবে এবং পড়া হলে স্থায়ীভাবে মুছে যাবে, অথবা মেয়াদ শেষ হলে স্বয়ংক্রিয়ভাবে মুছে যাবে। মেয়াদোত্তীর্ণ মেমো প্রতি ঘন্টায় স্বয়ংক্রিয়ভাবে পরিষ্কার করা হয়।
