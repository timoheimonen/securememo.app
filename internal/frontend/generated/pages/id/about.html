Tentang securememo.app

securememo.app adalah aplikasi yang berfokus pada privasi yang memungkinkan Anda membuat memo terenkripsi yang menghancurkan diri setelah dibaca atau kedaluwarsa. Dibangun dengan keamanan dan privasi sebagai prioritas utama. Kode sumber tersedia di GitHub.

Tumpukan Teknologi

  • Go service: Go service: Self-hosted HTTP service
  • SQLite database: SQLite database: Database bertenaga SQLite untuk penyimpanan aman
  • Web Crypto API: Web Crypto API: Enkripsi AES-256 sisi klien
  • HTML/CSS/JavaScript: HTML/CSS/JavaScript: Antarmuka web modern dan responsif
  • Kode sumber tersedia di GitHub: Lihat kode sumber di GitHub

Fitur Keamanan

🔐 Enkripsi Sisi Klien

Semua enkripsi terjadi di browser Anda menggunakan Web Crypto API. Memo Anda dienkripsi dengan AES-256 sebelum dikirim ke server kami.

🔑 Berbagi Kata Sandi Terpisah

Kata sandi enkripsi dihasilkan terpisah dari URL dan harus dibagikan melalui saluran yang berbeda untuk keamanan yang lebih baik.

💥 Memo yang Menghancurkan Diri

Setelah memo dibaca atau kedaluwarsa, memo langsung dan permanen dihapus dari database kami. Tidak ada jejak yang tersisa.

🌐 Tanpa Penyimpanan Kata Sandi

Kami tidak pernah menyimpan kata sandi enkripsi di server kami. Kata sandi hanya ada di memori browser Anda.

⚡ Performa Global

Dibangun di jaringan the self-hosted deployment untuk akses super cepat dari mana saja di dunia.

🔒 Privasi Utama

Tidak memerlukan akun pengguna. Tanpa pelacakan. Tanpa analitik. Privasi Anda adalah prioritas kami.

Cara Menggunakan

1. Buat Memo

Pergi ke "Buat Memo Aman" dan ketik memo Anda. Pilih waktu kedaluwarsa (8j, 1h, 2h, 1 minggu atau maksimal 30 hari). Memo dihapus saat dibaca atau kedaluwarsa. Sistem akan menghasilkan kata sandi enkripsi acak.

2. Bagikan Tautan dan Kata Sandi

Salin URL dan kata sandi yang dihasilkan, lalu bagikan kepada penerima melalui saluran terpisah untuk keamanan yang lebih baik.

3. Hancur Sendiri

Ketika penerima membuka tautan dan memasukkan kata sandi secara terpisah, memo akan didekripsi dan kemudian dihapus secara permanen jika dibaca, atau dihapus otomatis jika kedaluwarsa. Memo yang kedaluwarsa dibersihkan otomatis setiap 1 jam.
